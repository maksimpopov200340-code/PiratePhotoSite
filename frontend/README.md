# Frontend

Полноценный `React + TSX` фронт для текущего FastAPI API.

## Что уже есть

- сплошная лента постов
- красивое отображение картинок в карточках
- просмотр картинки в lightbox
- выбор текущего пользователя
- создание нового пользователя
- создание поста с текстом и картинкой
- просмотр комментариев прямо под постом
- отправка новых комментариев

## Запуск

```bash
cd frontend
npm install
npm run dev
```

По умолчанию `Vite` проксирует `/api` и `/media` на `http://127.0.0.1:8000`.
